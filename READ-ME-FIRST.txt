CARBON-12 RELAY 0.2.2 — DRAG & DROP UPDATE
==========================================

WHAT THIS IS
  A drop-in update that (1) lets you set the logbackups folder yourself,
  (2) makes "Catch up" REPORT exactly what it scanned so we can see why it
  found 0, and (3) carries the validated blueprint parser.

HOW TO APPLY (drag & drop)
  1. CLOSE the relay completely (tray icon -> Quit). Important: files lock if it's running.
  2. Open your relay project folder (the one with package.json, src\, electron\).
  3. From THIS folder, drag the "src" and "electron" folders INTO your project folder.
  4. Windows asks to replace files -> YES / "Replace the files in the destination".
     (This overwrites src\config.mjs, src\backfill.mjs, src\parser.mjs, etc. and
      electron\main.cjs. Nothing else is touched.)
