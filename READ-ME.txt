RELAY 0.2.5 — visual restyle (ops-console look)
===============================================
Full redesign of the app window in the Carbon-12 / Halo-UNSC aesthetic:
  • Telemetry status bar with live pulse + scanline (the signature)
  • Haul shown as a glowing instrument readout with category bars + recent list
  • Numbered ops modules, mono data readouts, disciplined color
  • Orange reserved for Stop/alerts only; cyan = synced/good
  • Settings (log path, launch-at-login) collapsed into a drawer
  • Version badge + logbackups folder field wired in

IMPORTANT: this is a DROP-IN replacement of electron/index.html. Every existing
function is preserved (pairing, start/stop, catch-up + diagnostics, haul loader,
health, digest, launch-at-login, logbackups override). No back-end changes.

APPLY (drag & drop)
  1. Close the relay (tray -> Quit).
  2. Drag "electron" into your project folder -> Replace index.html + preload.cjs -> Yes.
  3. Ship:  npm version patch  then  git push --follow-tags   (publish the draft release)
     or test now:  npm run dist  -> install.

NOTE ON THE HAUL CATEGORY BARS
  The bars (Armor/Weapons/etc.) show only if the haul endpoint returns a
  `categories` object. Right now it returns count + recent, so you'll see the big
  number + recent list immediately; the category bars light up once we add the
  category breakdown to the bot's /haul response (small follow-up, r255).
