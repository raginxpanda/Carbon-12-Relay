RELAY 0.2.21 — five quality-of-life additions
=============================================
Drag electron/ in -> Replace all -> Yes. Consolidated electron set (includes all
prior electron fixes: allowlist, update button + auto-popup + close-fix, live
count, recent-ordering, collapsible modules).

1) COLLAPSE-STATE MEMORY
   Every module section (Paired orgs, Catch up, Connection, Kill feed, Activity,
   Settings) now remembers whether you left it open or closed, across restarts.
   Set it up how you like once; it stays that way.

2) "COPIED!" CONFIRMATION
   The Copy button in Activity now flashes "Copied!" in cyan for a moment so you
   know the log actually copied.

3) LIVE SESSION SUMMARY
   A line under the status bar shows "N blueprints · N kills · N deaths · this
   session", updating live as events come in. Resets when you start a fresh session.
   (Kills/deaths fill in once combat data flows — pairs with the combat parser.)

4) KILL-FEED COUNT BADGE
   The Kill feed header shows a small orange badge with the session kill count, so
   even when that section is collapsed you can see combat happened. Hidden at 0.

5) MINIMIZE-TO-TRAY TOGGLE (Settings)
   New checkbox: "Keep running in the tray when I close the window." On by default
   (current behavior). Uncheck it and closing the window quits the app instead.

All prefs are stored in your relay config (with getUI/setUI), so they persist.
