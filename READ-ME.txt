RELAY 0.2.18 — COMBAT PARSER (the big one: kill feed / leaderboard / Nemesis go live)
=====================================================================================
Pairs with BOT r266. This is the piece that was blocked for weeks — now unblocked,
because a real kill-line format got confirmed (cross-checked against an independent
working kill-feed tool, citizenmon).

WHAT'S NEW
 • NEW FILE src/combat-parser.mjs — parses SC kill lines:
     <Actor Death> CActor::Kill: 'victim' ... killed by 'killer' ... using 'weapon'
       ... with damage type 'type'
   plus vehicle destruction (CVehicle::OnAdvanceDestroyLevel). Emits a NEUTRAL
   actor_death event (killer + victim). It does NOT decide who "won" — the BOT does
   that by comparing against your paired handle (correct, since only the bot knows
   which handle owns this device). Tags obvious NPCs so they don't inflate PvP.
 • src/relay.mjs — runs the combat parser alongside the location parser.
